# Simple Sales App

A simple sales management application built with Agentlang that integrates with HubSpot CRM. This app helps you manage leads, create deals, and automate sales workflows using HubSpot's API.

## Features

- **Automatic HubSpot Sync**: Leads and Deals are automatically synced to HubSpot when created locally
- **Lead Management**: Create and track sales leads locally and in HubSpot
- **Contact Sync**: Sync contacts from HubSpot to your local database
- **Deal Creation**: Create and manage deals in HubSpot
- **Lead Scoring**: Automatically score leads based on attributes
- **AI Sales Assistant**: AI-powered agent to help with sales operations

## Prerequisites

- HubSpot account with API access
